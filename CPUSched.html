Algorithms
All 6 scheduling algorithms

Each algorithm models a different CPU scheduling strategy used in real operating systems.

FCFS
First come, first served. Processes run in arrival order without interruption.
Non-preemptive
SJF
Shortest job first. Picks the process with the smallest burst time next.
Non-preemptive
SRT
Shortest remaining time. Preempts if a shorter process arrives mid-execution.
Preemptive
Round Robin
Equal time slices (quantum) for each process in a fair cyclic queue.
Preemptive
Priority
Highest priority process runs first. Supports preemptive and non-preemptive.
Both modes
Priority + RR
Priority groups first, then Round Robin is applied within each group.
Hybrid
